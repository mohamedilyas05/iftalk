# 📖 About IFtalk

IFtalk is a modern real-time messaging application designed to provide a seamless communication experience.

It focuses on:

- ⚡ Instant messaging
- 🌐 Real-time updates
- 🟢 Online user status
- 🕒 Last seen tracking
- 🖼️ Profile management
- 📱 Responsive design

---

# ✨ Features

## 🔐 Authentication

- User Registration
- User Login
- JWT Authentication
- Protected Routes
- Password Encryption

## 💬 Messaging

- One-to-One Chat
- Real-Time Message Delivery
- Dynamic Chat Window
- Instant Synchronization

## 🌍 Presence System

- Online Status
- Last Seen Updates
- User Activity Tracking

## 🖼️ Profile Management

- Profile Pictures
- Cloudinary Uploads
- User Profiles

## 🎨 UI Experience

- Responsive Layout
- Modern Design
- Smooth Interactions

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/mohamedilyas05/iftalk.git
```

## Frontend

```bash
cd app

npm install

npm run dev
```

## Backend

```bash
cd server

npm install

npm run dev
```
